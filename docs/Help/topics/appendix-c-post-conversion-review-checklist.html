Appendix C. Post-Conversion Review Checklist

  1. Did the forms open without reader errors?
  1. Did the project compile on the first pass?
  1. Did the application start and close normally?
  1. Did core workflows run correctly?
  1. Are the remaining issues structural, runtime, or merely visual?
  1. Was the converter fixed globally if a reusable defect was found?
